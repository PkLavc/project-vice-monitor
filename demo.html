Project Vice Monitor

Sistema de Monitoramento Estático - Sem necessidade de servidor

🎯 Sistema Completo

Este é um sistema de monitoramento totalmente estático que funciona sem necessidade de servidor web.

SISTEMA PRONTO

Funcionalidades Principais

  • Monitoramento em tempo real de 4 serviços GTA VI
  • Detecção automática de mudanças de conteúdo (Intelligence Update)
  • Geração automática de blog entries para atualizações
  • Dashboard HTML totalmente estático
  • Blog com links diretos (sem carregamento via JavaScript)
  • Gráficos de performance e SLA
  • Histórico de incidentes
  • Sistema de badges e estatísticas

🚀 Como Funciona

1. Monitoramento (monitor_new.py)

Executa verificações automáticas dos serviços e gera dados HTML:

python monitor_new.py

Resultado: Atualiza index.html e blog.html automaticamente

2. Dashboard Estático (index.html)

Funcionalidades: Dados integrados, gráficos, estatísticas

Funcionalidades: Relatórios automáticos, links diretos
